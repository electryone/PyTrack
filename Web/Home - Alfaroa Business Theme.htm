The Simplest Computer Vision Tracking Tool

PyTrack is a lightweight and well documented codebase that allows you to do simple computer vision tracking with Python

Download Now   Learn More

Python Powered

PyTrack is built on Python 3.2 and PyGame, rather than complicated libraries. This makes it really easy to use, modify, and extend without being an expert programmer.

Free and Open Source

No reason to pay for expensive commercial software, PyTrack is free for your use and tinkering. Feel free to suggest features, and help us develop.

Cross Platform

You can use PyTrack on Windows, Mac, Linux and dozens of other platforms that PyGame supports. We will have it running on a toaster at some point too.

Fast

PyTrack's dependencies use optimized C, and Assembly code for core functions. We have got up to 120 frames/sec on post processing, and are still improving.

Extensible

We work hard to create great documentation and demos, to make creating your own PyTrack based computer vision project as easy as possible.

Research Based

Born in a University of Washington lab, PyTrack was created to add simplicity to basic computer vision for biological experiments.
